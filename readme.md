# Document ready

> Simple document ready listener

## Install

```
$ npm install @document/ready
```


## Usage

```js
const documentReady = require('@document/ready');

documentReady(() => {
  // work your magic
});
```

## API

### documentReady(callback)

#### callback

Type: `function`

Will be executed when the document is ready.
